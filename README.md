# 博物馆文物修复工作室 🏛️

一款桌面客户端游戏，玩家扮演文物修复师，通过观察、清洁、拼合和布展完成不同年代藏品的修复任务。

## 游戏特色

### 7个功能窗口
- **主菜单** - 游戏入口，图鉴收藏展示
- **修复台** - 核心玩法：观察、清洁、拼合、补配、上色、封护
- **材料柜** - 库存管理、材料商店、材料调配、工具解锁
- **检测室** - 年代检测、材质分析、X光扫描
- **任务档案** - 可接委托、当前任务、修复记录
- **展厅布置** - 展柜安排、观众反馈、展厅评级
- **学徒培训** - 招聘学徒、技能培训、助手管理

### 核心玩法
- 📋 选择委托任务，查看破损报告
- 🔍 放大观察文物纹理，发现破损点
- 🧹 清理污渍和锈蚀
- 🧩 拼接破碎的文物碎片
- 🎨 调配颜料进行全色修复
- ⚖️ 控制工具力度，避免二次损伤
- 🔬 进行年代和材质科学检测
- 📝 记录修复步骤
- 💰 管理预算和耗材
- 🔧 解锁高级修复工具
- 👨‍🎓 培训学徒助手
- 🖼️ 安排展柜位置
- 💬 收集观众反馈
- ⭐ 获得评级与图鉴收藏

## 技术栈
- Electron - 桌面应用框架
- HTML5 + CSS3 + JavaScript - 前端技术
- Node.js - 后端运行时

## 运行方式

### 安装依赖
```bash
npm install
```

### 启动游戏
```bash
npm start
```

## 游戏提示
1. 先从任务档案接受一个委托
2. 前往修复台开始修复工作
3. 按步骤完成：观察 → 清洁 → 拼合 → 补配 → 上色 → 封护
4. 修复完成后可以在展厅布置中展出
5. 用获得的奖励购买材料和解锁工具
6. 招聘学徒来协助你的工作

祝你修复愉快！🎮
